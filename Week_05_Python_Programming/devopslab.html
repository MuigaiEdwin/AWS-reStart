Jenkins - CI/CD Automation

What it does: Automates building, testing, and deploying code through pipelines.

  • Auto-build and test on code commits
  • Deploy applications automatically
  • Run tests across multiple platforms
  • Schedule automated tasks and workflows

AWS CodeDeploy - Deployment Service

What it does: Automates deployments to EC2, Lambda, and on-premises servers.

  • Zero-downtime deployments (blue/green)
  • Multi-region deployments
  • Lambda function updates with traffic shifting
  • Hybrid cloud deployments

Lab 3: Automation vs Orchestration

Think like a programmer: Automation = one function; Orchestration = coordinated workflow.

Automation

Single tasks (scripts, functions)

def backup():
  ...
Orchestration

Multiple tasks coordinated

workflow.execute()

Key Differences

Automation: Automates a single repeatable task

  • Python scripts
  • Unit tests
  • Cron jobs
  • User-defined functions

Orchestration: Coordinates multiple automated tasks into workflows

  • Terraform - Infrastructure provisioning
  • Kubernetes - Container orchestration
  • Workflow engines
  • Process coordination

Key Takeaways

  • Value of Automation: Speeds delivery, reduces errors, increases consistency
  • Not Everything Should Be Automated: Some tasks require human judgment or creativity
  • Start Small: Automate repetitive tasks first, then orchestrate workflows
  • Engineer Mindset: Break problems into functions (automation) then compose into systems (orchestration)
